##Описание программы

Currency Converter - это GUI-приложение для конвертации валют с использованием актуальных курсов из внешнего API. Приложение позволяет конвертировать суммы между различными валютами, сохранять историю конвертаций в формате JSON, загружать историю при запуске, а также удалять отдельные записи из истории.
Возможности:

Выбор валюты "из" и валюты "в" из списка доступных валют


Ввод суммы для конвертации с проверкой корректности


Получение актуальных курсов валют через внешний API (exchangerate-api.com)


Отображение результата конвертации


Сохранение истории конвертаций в JSON-файл


Автоматическая загрузка истории из JSON-файла при запуске


Удаление всей истории конвертаций


Валидация вводимых данных (сумма должна быть положительным числом)

Инструкция по запуску
Требования:

Python 3.6 или выше


Библиотека requests (устанавливается командой: pip install requests)


Установленная библиотека Tkinter (обычно идёт в комплекте с Python)

Запуск:

Откройте терминал или командную строку


Перейдите в директорию проекта:

bash

cd currency_converter


Запустите приложение:

bash

python main.py

Примеры использования
Конвертация валюты:

Введите сумму в поле "Amount" (например, 100)


Выберите исходную валюту из выпадающего списка "From Currency" (например, USD)


Выберите целевую валюту из выпадающего списка "To Currency" (например, EUR)


Нажмите кнопку "Convert"


Результат конвертации отобразится под кнопкой (например, "100.00 USD = 93.50 EUR")

Работа с историей:

Каждая успешная конвертация автоматически добавляется в таблицу истории


История содержит дату, сумму, исходную валюту, целевую валюту и результат


Для очистки всей истории нажмите кнопку "Clear History" и подтвердите действие


При следующем запуске приложения история автоматически загрузится из файла

Валидация данных:
Приложение проверяет корректность ввода суммы:

Поле суммы не может быть пустым


Сумма должна быть числом (целым или дробным)


Сумма должна быть больше нуля


При ошибке ввода под полем появляется подсказка с пояснением

Тестовые данные для проверки:
Попробуйте выполнить следующие операции для тестирования:
Конвертация 50 долларов США в евро:

Сумма: 50


Из: USD


В: EUR

Конвертация 1000 рублей в доллары США:

Сумма: 1000


Из: RUB


В: USD

Конвертация 75 евро в британские фунты:

Сумма: 75


Из: EUR


В: GBP

Проверка валидации:

Попробуйте ввести отрицательное число (например, -10)


Попробуйте ввести буквы вместо цифр


Попробуйте оставить поле суммы пустым

Проверка истории:

Выполните несколько конвертаций


Закройте приложение и откройте снова


Убедитесь, что история сохранилась


Нажмите "Clear History" и подтвердите удаление

Использование API
Приложение использует бесплатный API сервиса ExchangeRate-API (exchangerate-api.com). Для базового использования API-ключ не требуется - достаточно 1500 запросов в месяц. При необходимости большего количества запросов можно зарегистрироваться на сайте сервиса и получить бесплатный ключ, после чего модифицировать код для его использования.
Формат запроса к API:

Базовый URL: https://api.exchangerate-api.com/v4/latest/USD


Ответ содержит курсы всех валют относительно базовой


Приложение автоматически получает курс для выбранной пары валют

Используемые технологии

Python 3 - язык программирования


Tkinter - стандартная GUI-библиотека Python


Requests - библиотека для HTTP-запросов к API


JSON - формат хранения истории конвертаций


Git - система контроля версий

Примечания по работе

Для работы приложения требуется активное интернет-соединение для получения курсов валют


При отсутствии интернета приложение покажет сообщение об ошибке


Список доступных валют загружается из API при запуске приложения


При проблемах с загрузкой списка валют используется стандартный набор из пяти основных валют
